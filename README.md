# 🛒 Customer Shopping Behavior Analysis

## 📌 Project Overview

This project focuses on analyzing customer shopping behavior using **SQL, Python (Pandas), and Power BI**. The goal is to extract meaningful insights from raw data to help businesses understand customer preferences, purchasing patterns, and product performance.

---

## 🎯 Objectives

* Analyze customer purchase trends
* Identify top-performing products and categories
* Understand customer segmentation
* Improve decision-making using data insights

---

## 🛠️ Tools & Technologies Used

* **SQL (MySQL)** – Data extraction and querying
* **Python (Pandas, NumPy)** – Data cleaning and analysis
* **Power BI** – Data visualization and dashboard creation

---

## 📂 Dataset

* Source: Kaggle / Provided dataset
* Contains customer transactions, product details, and purchase frequency

---

## 🔄 Project Workflow

1. **Data Collection**
   Imported dataset into MySQL database

2. **Data Cleaning (Python - Pandas)**

   * Handled missing values
   * Converted data types
   * Created new features (e.g., purchase frequency in days)

3. **Data Analysis (SQL + Python)**

   * Top selling products
   * Category-wise performance
   * Customer purchase patterns

4. **Data Visualization (Power BI)**

   * Interactive dashboards
   * KPI metrics
   * Trend analysis

---

## 📊 Key Insights

* Identified top 3 most purchased products in each category
* Found customer purchase frequency patterns
* Highlighted high-revenue generating categories

---

## 📈 Dashboard Features

* Sales overview
* Category performance
* Customer segmentation
* Time-based trend analysis

---

## 🚀 How to Run the Project

1. Clone the repository
2. Import dataset into MySQL
3. Run SQL queries
4. Execute Python scripts
5. Open Power BI dashboard file
